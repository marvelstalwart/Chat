# Chat
<h3>A simple web messaging with 1-1 and Group chat and Video calling, implemented with Simple Peer WEB RTC and Web socket</h3>

<h5>FEATURES</h5>
<ul>
  <li>Real Time Messaging</li>
  <li>Video Call</li>
  <li>1 to 1 and Group Chat</li>
  <li>Avatar Customization</li>
  <li>Online/Offline Status</li>
  <li>Add/Remove Group Members</li>
  <li>Add/Dismiss Admin</li>
<li>Advanced search</li>

</ul>
